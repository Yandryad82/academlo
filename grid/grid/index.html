CSS GRID

Grid me permite generar grillas o cuadriculas compuestas por filas y columnas

  • Permite generar filas (horizontales) y asignarles altura
  • Permite generar columnas (verticales) y asignarles anchura
  • Podemos generar espaciado entre los elementos grid
  • Las lineas nos permiten ubicar a los elementos y posicionarlos
